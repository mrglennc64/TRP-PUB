What Roya checks automatically

Upload your metadata file once. Roya runs all four checks simultaneously and surfaces every issue in a single guided review.

🩺
Metadata Health Audit
Checks every field for completeness, invalid formats, and missing required values. Calculates a quality score per work.
ISWC IPI Publisher Format validation
⚖️
Split Verification
Validates that ownership splits sum to exactly 100% for every work. Flags conflicts, missing contributors, and rounding errors.
Split totals Contributor roles Rounding

Identifier Gap Detection
Finds missing or malformed ISRCs, ISWCs, and IPI numbers. Prioritizes works most likely to cause royalty routing failures.

Duplicate Detection
Uses fuzzy title matching to surface potential duplicate works across your catalog — before they create split disputes or double-registration conflicts.

Run individual checks

Each tool can also be used standalone for targeted analysis.

1. Metadata Health Audit
Checks completeness, missing fields, invalid formats, and metadata quality score.
Try Health Audit
2. ISRC / Identifier Gap Detection
Detects missing or invalid ISRCs, IPIs, and other required identifiers.
Check Identifiers
3. Metadata Reconciliation
Finds duplicate works across datasets and returns confidence-based matches.
Run Reconciliation
4. Split Verification
Validates split totals and flags missing, conflicting, or inconsistent entries.
Verify Splits

Summarizes overall submission readiness — READY / NEEDS REVIEW / NOT READY — with recommended fixes.
Check Readiness

Matches recordings to compositions using Jaro-Winkler and cosine similarity. Identifies unlinked royalty flows.
