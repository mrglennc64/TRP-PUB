How Roya Works

A simple pre-submission verification layer for publishers

No integration required
No workflow change
PRO remains final authority

1

Publisher uploads PRO registration data

Using their existing workflow, publishers export their PRO CSV files (exactly as they do today) and upload them to Roya.

ðŸ“¤ What they upload:

  • PRO CSV export from their catalog system
  • Split sheets with contributor names, roles, percentages
  • IPI/ISWC identifiers (if available)

PRO CSV

Same format. Same system. No changes.

2

Roya validates and detects issues

Roya automatically checks for common errors that cause PRO rejections: missing IPI numbers, split totals that don't add up, invalid formats, and potential ownership conflicts.

ðŸ” What we check:

  • Split percentages total 100%
  • IPI/ISWC identifiers present and valid
  • Contributor names not missing
  • Role types consistent

Auto-detects errors

Returns clear report of issues found

3

Auto-fix and verification record

For common issues, Roya can automatically correct them (e.g., normalizing percentages to 100%). All verifications receive a timestamped audit record.

✨ Auto-fix capabilities:

  • Normalize split totals to 100%
  • Generate missing IPI placeholders
  • Flag potential conflicts for review

Verification Record

Timestamped audit trail

4

Publisher submits clean data to PRO

After fixing any issues, publishers download the verified file and submit it to PRO using their normal process — now with confidence it will process without delays.

✅ Result:

  • Fewer rejections
  • Faster processing
  • Audit trail for disputes
  • Same PRO workflow

Ready for PRO

Clean data, same submission process

Quick Answers

Does this change PRO workflow? No — publishers still submit the same way. Roya is an optional pre-check.
What if we miss something? PRO's validation is still final. Roya just catches issues earlier.
Do publishers need to install software? No — it's a web-based tool. Upload, verify, download.
How long does verification take? Seconds. Publishers get instant feedback.
